Changes to upload to Google Play

RESPECTING THE INTELECTUAL PROPERTY. Our app contents logotypes such as the one from the DGT (We are forbidden to use it).

SECURITY AND ENCRYPTION for users login. It is required to improve it in the server as well as the database encryption, because a hacker can inject a MySQL code and get the keys.

IMPROVE ARCHIVES string.xml In this archive we indicate the text in order to translate the language of whatever device

LAYOUTS FOR DIFFERENT SCREEN TYPES We have to adapt them to all screen sizes. Nowadays the app only distinguishes between land and portrait. It occurs something similar with images, because we have to add them with different resolutions in order to visualize them accurately in all the screens.

Generate content in other language

We can add the translation of the information of Google Play to the app, clicking in "Manage Translation" near the upper part of the page called "Play Store card"

If the translation is not done, it can be translated automatically by the Google Translator or being visualize in the native language of the app

In order to comercialize the app in differents countries, Google recommends adding the screenshots and the video in differents languages.

According to the language preferences selected by each user in their Android devices, the videos will appear in one or another language
