Vehicle Registration

Once we have entered with the correct user, a window appears for vehicle registration. Here we incorporate the data of the car that is indicated below.

We have to add the chassis number strictly of 17 digits, that do not have to be numbers necessarily

If the chassis number is incorporated correctly, the vehicle will be added in the database and it will show as a Toast message indicated in

On the contrary, it shows an alert wich says that the chassis number is invalid and it allows us to write the number again

When it is incorporated to the database, the layout is refreshed. In this way, another car can be added instantaneously

Parameters to add:
  • *Chassis number
  • *Car Number
  • *Metric Horsepower
  • *Model
  • *Owner
  • *CC
  • *Brankd
  • *Combustible
