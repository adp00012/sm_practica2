Login User

Once the user has iniciated the app with the previous layout button the login form can be seen. Here we can inserted the parameters that are highlighted below.

The application takes over the errors of each one of the incorretly inserted parameters.

If the login is correct, you can continue with the following vehicles' registration layout

It can be needed an users registration section in the application, but it is not specified that its usage is limited by professionals of the DGT and therefore, the administrator is in charge of registring users. In this way, the usage will be restricted to any external person from administration.

Parameters to incorporate:
  • *IP Server Direction
  • *Server Port
  • *Username
  • *Password
